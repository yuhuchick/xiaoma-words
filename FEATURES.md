# 小马单词 - 功能特性说明

## 🎯 核心功能

### 1. 智能单词学习系统

#### 新单词学习
- **智能推荐**: 根据用户水平和学习进度推荐合适的新单词
- **分类学习**: 支持按类别学习（基础词汇、商务英语、学术英语等）
- **难度分级**: 单词按难度分为1-3级，循序渐进
- **发音支持**: 每个单词都配有音标和发音功能

#### 间隔复习系统
- **科学算法**: 采用间隔重复算法，根据记忆曲线安排复习
- **复习提醒**: 自动提醒需要复习的单词
- **进度跟踪**: 实时跟踪每个单词的掌握程度
- **自适应调整**: 根据答题正确率调整复习频率

### 2. 丰富的单词数据

#### 单词信息完整
- **多词性释义**: 支持单词的多种词性和含义
- **例句展示**: 每个释义都配有英文例句和中文翻译
- **同义词反义词**: 帮助理解单词的语义关系
- **词源信息**: 了解单词的历史渊源
- **用法说明**: 指导单词的正确使用场景

#### 分类体系
- **基础词汇** (500词): 日常生活常用词汇
- **商务英语** (300词): 职场商务相关词汇
- **学术英语** (400词): 学术研究相关词汇
- **旅游英语** (250词): 旅行出行相关词汇
- **科技英语** (350词): 科技技术相关词汇

### 3. 个性化学习体验

#### 学习进度管理
- **掌握等级**: 0-3级掌握程度评估
- **学习历史**: 详细记录每次学习情况
- **准确率统计**: 实时计算学习准确率
- **学习时长**: 记录总学习时间和单次学习时长

#### 学习建议系统
- **智能推荐**: 根据学习情况推荐学习内容
- **目标提醒**: 提醒完成每日学习目标
- **复习提醒**: 提醒需要复习的单词
- **进度分析**: 分析学习趋势和改进建议

### 4. 数据管理架构

#### 独立数据文件
- **words.json**: 单词数据库，包含完整的单词信息
- **user-study.json**: 用户学习记录和进度数据
- **wordManager.js**: 数据管理工具类，提供统一的数据操作接口

#### 本地存储支持
- **数据持久化**: 学习进度自动保存到本地存储
- **离线学习**: 支持离线学习，数据同步到云端
- **数据备份**: 支持数据导出和恢复

#### 后端对接准备
- **标准化接口**: 数据格式标准化，便于后端API对接
- **用户系统**: 支持用户登录和数据同步
- **云端存储**: 学习数据可同步到云端

## 🔧 技术特性

### 1. 数据管理
```javascript
// 获取单词数据
const wordData = wordManager.getWordById('w001')

// 更新学习进度
wordManager.updateWordProgress('w001', 'correct', 'new')

// 获取学习建议
const recommendations = wordManager.getStudyRecommendations()
```

### 2. 学习算法
- **间隔重复**: 1天、3天、7天、14天、30天、90天复习间隔
- **掌握等级**: 根据答题正确率动态调整掌握等级
- **难度自适应**: 根据用户表现调整学习难度

### 3. 用户体验
- **实时反馈**: 学习过程中提供即时反馈
- **进度可视化**: 直观显示学习进度和统计
- **操作便捷**: 简化的操作流程，降低学习门槛
